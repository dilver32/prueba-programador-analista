# Instrucciones de la prueba técnica
La prueba consta de componentes prácticos distribuidos en las carpetas principales:
 

## 1. Desarrollo
Dentro de la carpeta `Desarrollo/` deberás crear e implementar el proyecto en Laravel con su solución planteada.
Asegúrate de incluir un archivo `.env.example` y documentar brevemente cualquier instrucción adicional de configuración en caso de ser necesario.


## 2. Base de Datos y Consultas
En la carpeta SQL, se encuentra la base de datos de prueba Chinook (`Chinook.sql`), deberás montar y restaurar esta base de datos utilizando el entorno de desarrollo local Laragon.
Desarrolla las tres (3) consultas SQL solicitadas en el enunciado de la prueba.
La solución a las consultas debe colocarse en un script independiente (por ejemplo, `soluciones/consultaN.sql`) dentro de la misma carpeta `SQL/`.
